# Project: Fashion Blog

## Overview

In this project, you will build the **HTML structure of a fashion blog**
for a fictional blogger named **Isa**.

The goal is to apply the HTML concepts you have learned to create a
basic multi-section webpage.

The blog should include:

- headings
- paragraphs
- images
- links
- lists
- structured sections

This project reinforces **HTML fundamentals** used in real websites.

---

## Project Scenario

Isa is a fashion blogger preparing for **New York Fashion Week**.

She needs a website where her followers can:

- read her blog posts
- view fashion photos
- navigate the page easily
- explore related links

Your task is to build the **HTML structure** of the blog.

---

## Key HTML Elements to Use

The project should include many of the elements learned in the course.

### Headings

Used to organize sections of the blog.

Example:
```Text
    <h1>Isa's Fashion Blog</h1>
    <h2>New York Fashion Week</h2>
```
---

### Paragraphs

Used to display text content.

Example:

    <p>Welcome to my fashion blog where I share the latest trends.</p>

---

### Images

Images showcase fashion designs and outfits.

Example:

    <img src="fashion.jpg" alt="Fashion runway outfit">

Structure:
```Text
    img
     ├── src
     └── alt
```
---

### Links

Links allow users to navigate to:

- external websites
- fashion references
- social media pages

Example:

    <a href="https://example.com">Read More</a>

---

### Lists

Lists help organize information.

Example:
```Text
    <ul>
        <li>Runway Shows</li>
        <li>Street Fashion</li>
        <li>Designer Interviews</li>
    </ul>
```
---

## Example Blog Structure

Example HTML layout:
```Text
    <!DOCTYPE html>
    <html>
      <head>
        <title>Isa's Fashion Blog</title>
      </head>
      <body>

        <h1>Isa's Fashion Blog</h1>

        <p>Welcome to my fashion blog covering New York Fashion Week.</p>

        <img src="fashion.jpg" alt="Fashion runway show">

        <h2>Top Trends</h2>

        <ul>
          <li>Bold colors</li>
          <li>Oversized jackets</li>
          <li>Vintage styles</li>
        </ul>

        <a href="https://fashionweek.com">Visit Fashion Week</a>

      </body>
    </html>
```
---
