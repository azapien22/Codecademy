# Introduction to HTML

## Overview
HTML (HyperText Markup Language) is the standard language used to create and structure web pages.

It is often considered the **skeleton of a website** because it defines the structure of the content displayed in the browser.

Common content structured by HTML includes:
- Text
- Images
- Videos
- Links
- Other media elements

You can view the HTML of any webpage by:
1. Right-clicking on the page
2. Selecting "Inspect"
3. Viewing the HTML inside the Developer Tools panel

---

## Why Learn HTML

HTML is usually the **first language learned in web development**.

It is widely used by:
- Developers
- Designers
- Marketers
- Front-end engineers

Learning HTML allows you to:
- Build websites
- Structure page content
- Modify templates for blogs or newsletters
- Understand how websites function

HTML is a **core technology of the web**.

---

## What Does HTML Stand For?

HTML = **HyperText Markup Language**

### Markup Language

A markup language is a computer language used to define the **structure and presentation of text**.

HTML wraps raw text in **elements** so browsers know how to display the content.

Example:

    <p>This is a paragraph</p>

The `<p>` tag tells the browser that the text is a **paragraph element**.

---

## HTML Elements

HTML documents are made up of **elements**.

Most HTML elements contain:

- Opening tag
- Content
- Closing tag

Example:

    <h1>Hello World</h1>

Breakdown:

Opening Tag:
    <h1>

Content:
    Hello World

Closing Tag:
    </h1>

Elements are the **building blocks of web pages**.

---

## HyperText

HyperText refers to text that links to other resources.

These connections are called **hyperlinks**.

Hyperlinks allow users to:
- Navigate between pages
- Access other websites
- Jump to different sections of content

Example:

    <a href="https://www.codecademy.com">Visit Codecademy</a>

---

## Types of Content HTML Can Display

HTML can structure many kinds of content including:

- Headings
- Paragraphs
- Images
- Videos
- Links
- Lists
- Tables

Example:

    <h1>Main Heading</h1>
    <p>This is a paragraph.</p>
    <img src="image.jpg">
    <a href="https://example.com">Example Link</a>

---

## HTML with CSS and JavaScript

Modern websites typically combine three technologies:

HTML
    Provides structure and content

CSS
    Controls layout, colors, and design

JavaScript
    Adds interactivity and dynamic behavior

Simple analogy:

HTML = Skeleton  
CSS  = Appearance / styling  
JS   = Behavior / movement

---

## What You Will Learn First

When beginning HTML, the focus is on adding basic page content such as:

- Text
- Images
- Videos
- Links

Early pages may look simple or unattractive. This is normal because styling is handled later with **CSS** and functionality with **JavaScript**.

The primary goal at first is to understand **how content is structured in HTML**.
