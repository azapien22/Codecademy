# HTML Anatomy

## Overview
HTML webpages are built using **elements**.

Elements define:
- The structure of a webpage
- The type of content displayed

Examples of content:
- Text
- Images
- Videos
- Links
- Other HTML elements

---

## Basic HTML Element Structure

A simple HTML paragraph element looks like this:

    <p>Hello World!</p>

This element contains three parts:

1. Opening tag
2. Content
3. Closing tag

---

## Visual Structure

Example element:

    <p>Hello World!</p>

Breakdown:

Opening Tag
    <p>

Content
    Hello World!

Closing Tag
    </p>

Together these form an **HTML element**.

---

## HTML Element

Definition:

An **HTML element** is a unit of content in an HTML document.

It consists of:
- An opening tag
- Content
- A closing tag

Example:

    <p>Hello World!</p>

In this example:
- The element type is **paragraph**
- The content is **Hello World!**

---

## HTML Tag

An **HTML tag** is the element name surrounded by angle brackets.

Structure:

    <tagname>

Example:

    <p>

Tags tell the browser **how to interpret the content**.

---

## Opening Tag

The **opening tag** begins an HTML element.

Example:

    <p>

This tells the browser:
"Start a paragraph element."

Structure:

    <tagname>

---

## Content

The **content** is the information placed between the opening and closing tags.

Content can include:

- Text
- Images
- Videos
- Links
- Other HTML elements

Example:

    Hello World!

Inside element:

    <p>Hello World!</p>

---

## Closing Tag

The **closing tag** ends an HTML element.

Closing tags contain a **forward slash (/)**.

Example:

    </p>

Structure:

    </tagname>

This tells the browser:
"The paragraph element ends here."

---

## Complete Element Example

    <p>Hello World!</p>

Parts:

Opening Tag
    <p>

Content
    Hello World!

Closing Tag
    </p>

This entire structure is called an **HTML element**.

---

## Key Takeaway

HTML documents are made of **elements**.

Each element usually contains:

    Opening Tag
    Content
    Closing Tag

Example:

    <tagname>content</tagname>
