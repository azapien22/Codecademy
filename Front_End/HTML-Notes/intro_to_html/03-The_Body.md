# The Body Element

## Overview

One of the most important HTML elements used to build a webpage is the **body element**.

The **body** contains all the content that is displayed on the webpage.

Anything placed **inside the opening and closing `<body>` tags** will appear in the browser window.

Anything **outside the body** will not be displayed to the user.

---

## Body Tag Structure

The body element uses two tags:

    <body>
    </body>

- `<body>` → opening tag
- `</body>` → closing tag

Everything between these tags is visible on the page.
