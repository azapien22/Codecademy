# CSS — ID Selector

## Overview

Sometimes you need to style **one specific element**.

For this, HTML provides the:

    id attribute

The **ID selector** targets a single, unique element.

---

## HTML with ID

Example:

    <h1 id="large-title">Title</h1>

Structure:
```text
    element
     └── id="large-title"
```
---

## CSS ID Selector

To select an ID in CSS, use a **# (hash symbol)**.

Example:
```text
    #large-title {
      property: value;
    }
```
---

## Structure Diagram

    selector → #large-title
        |
        ▼
    targets → id="large-title"

---

## How It Works

HTML:

    <h1 id="large-title">Title</h1>

CSS:
```text
    #large-title {
      color: blue;
    }
```
Result:

    only this element becomes blue

---

## Key Rules

- starts with `#`
- must match the id exactly
- only one element per page should use the ID

---

## ID vs Class

### ID

    #large-title

- unique
- used once per page
- targets a single element

---

### Class

    .brand

- reusable
- used on multiple elements
- targets groups

---

## Example Comparison

HTML:

    <h1 id="title">Main Title</h1>
    <p class="text">Paragraph</p>

CSS:

    #title { color: red; }
    .text  { color: blue; }

---

## Important Limitations

- only one ID per element
- should not reuse the same ID multiple times
- each page should have unique IDs

---

## Visual Concept
```text
    page
     ├── id → unique element
     └── class → reusable styles
```
---

## Key Pattern

HTML:

    id="unique-name"

CSS:
```text
    #unique-name {
      property: value;
    }
```
---

## Key Idea

ID selectors are used for **unique, one-of-a-kind elements**.

    #id → single element styling
