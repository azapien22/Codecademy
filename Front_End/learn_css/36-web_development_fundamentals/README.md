# Project — Dasmoto's Arts & Crafts

## Project Overview

In this off-platform project, you will build a complete website for a fictional arts and crafts store called **Dasmoto's Arts & Crafts** using your own development environment.

Unlike previous Codecademy projects, this project provides minimal guidance. You are expected to apply everything you have learned so far and use external resources when necessary.

---

# Learning Objectives

This project is designed to reinforce your understanding of:

- Project organization
- HTML structure
- CSS styling
- External stylesheets
- CSS selectors
- Typography
- Images
- Colors
- Backgrounds
- Display properties
