# Project — The Box Model: Davie's Burgers

## Project Overview

In this project, you will apply your knowledge of the CSS Box Model to repair and improve a fictional restaurant website called **Davie's Burgers**.

The HTML structure and most of the CSS have already been created. Your task is to add and adjust Box Model properties to correctly size and space the elements on the page.

---

# Skills Practiced

Throughout this project, you will work with:

- width
- height
- padding
- border
- margin
- box-sizing

These properties control how elements occupy space and interact with surrounding content.

---

# Project Goal

Use CSS Box Model properties to:

- Control element dimensions
- Create spacing inside elements
- Create spacing between elements
- Add borders around content
- Improve layout consistency

# Box Model Review

Every HTML element consists of:

    +----------------------+
    |        Margin        |
    |  +----------------+  |
    |  |    Border      |  |
    |  | +------------+ |  |
    |  | |  Padding   | |  |
    |  | | +--------+ | |  |
    |  | | |Content | | |  |
    |  | | +--------+ | |  |
    |  | +------------+ |  |
    |  +----------------+  |
    +----------------------+

---

# Common Properties

## Width

Sets element width.

Example:

    width: 300px;

---

## Height

Sets element height.

Example:

    height: 200px;

---

## Padding

Space between content and border.

Example:

    padding: 20px;

---

## Border

Visible edge around an element.

Example:

    border: 2px solid black;

---

## Margin

Space outside an element.

Example:

    margin: 20px;

---

# Helpful Reminder

Modern projects often begin with:

    * {
      box-sizing: border-box;
    }

This keeps element sizing predictable.
