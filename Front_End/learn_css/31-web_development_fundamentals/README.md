# Project Notes — Olivia Woodruff Portfolio

## Project Overview

Project:

    Olivia Woodruff Portfolio

Goal:

    Use CSS Visual Rules to improve the appearance of a photography portfolio website.

Concepts Practiced:

- Background Color
- Text Alignment
- Font Size
- Opacity
- Font Weight
- Text Color
- Font Family
- Background Images

---

# Task 1 — Style Header Background

## Objective

Make the header section stand out.

## CSS Selector

    .header

## CSS
```text
    .header {
      background-color: CornflowerBlue;
    }
```
## Result

- header receives a blue background
- visually separates header from page content

---

# Task 2 — Center Header Text

## Objective

Center all text in the header.

## CSS
```text
    .header {
      text-align: center;
    }
```
## Result

- text centered horizontally
- cleaner page layout

---

# Task 3 — Increase About Me Font Size

## Objective

Make the About Me paragraph easier to read.

## CSS Selector

    .about-me

## CSS
```text
    .about-me {
      font-size: 20px;
    }
```
## Result

- larger paragraph text
- improved readability

---

# Task 4 — Add Transparency

## Objective

Blend the About Me section with the background.

## CSS
```text
    .about-me {
      opacity: 0.5;
    }
```
## Result

- paragraph becomes 50% transparent
- softer appearance

```text
    .header {

    .about-me {
      font-size: 20px;
      opacity: 0.5;
```

# Visual Rules Used

## Background Color

    background-color: CornflowerBlue;

Changes element background color.

---

## Text Align

    text-align: center;

Centers text horizontally.

---

## Font Size

    font-size: 20px;

Changes text size.

---

## Opacity

    opacity: 0.5;

Makes element semi-transparent.

---

## Font Weight

    font-weight: bold;

Makes text bold.

---

## Text Color

    color: Azure;

Changes text color.

---

## Font Family

    font-family: Georgia;

Changes typeface.

---

## Background Image

    background-image: url(...);

Places an image behind content.
