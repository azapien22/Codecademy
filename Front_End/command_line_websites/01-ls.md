# COMMAND LINE NAVIGATION — LS

===============================================================================
1. WHAT IS A COMMAND?
===============================================================================

A:

    command

is a directive that tells the computer to perform a specific task.

Basic process:

    USER
      |
      v
    TYPES COMMAND
      |
      v
    SHELL INTERPRETS COMMAND
      |
      v
    OPERATING SYSTEM PERFORMS TASK
      |
      v
    RESULT DISPLAYED


Example command:

    ls

The `ls` command asks the computer to display the contents of the current
directory.

===============================================================================
2. THE LS COMMAND
===============================================================================

The:

    ls

command is used to:

    LIST

the files and directories inside the current directory.

The name:

    ls

comes from:

    list


IMPORTANT:

Type:

    lowercase letter l

followed by:

    lowercase letter s


Correct:

    ls


Incorrect:

    1s


Remember:

    l = letter "L"

    1 = number "one"

===============================================================================
3. BASIC SYNTAX
===============================================================================

Command:

    ls


When shown with the shell prompt:

    $ ls


IMPORTANT:

The:

    $

symbol represents the shell prompt.

You normally type:

    ls

You do NOT type the `$` character itself.

===============================================================================
4. WHAT IS THE SHELL PROMPT?
===============================================================================

When working in a terminal, you may see:

    $

This is called the:

    shell prompt

The shell prompt indicates that the terminal is:

    ready to accept a command


Mental Model:

    $

        ↓

    SHELL IS READY

        ↓

    TYPE COMMAND

        ↓

    PRESS ENTER

        ↓

    COMMAND EXECUTES

        ↓

    OUTPUT APPEARS

        ↓

    PROMPT RETURNS

===============================================================================
5. LS EXAMPLE
===============================================================================

Suppose the current directory contains:

    2014/

    2015/

    hardware.txt


Command:

    $ ls


Output:

    2014    2015    hardware.txt


The `ls` command displays the contents of the current directory.

===============================================================================
6. UNDERSTANDING THE OUTPUT
===============================================================================

Output:

    2014    2015    hardware.txt


The current directory contains:

    2014/

        directory


    2015/

        directory


    hardware.txt

        file


Filesystem Diagram:

    current-directory/
    |
    +-- 2014/
    |
    +-- 2015/
    |
    +-- hardware.txt

===============================================================================
7. HOW LS WORKS
===============================================================================

When you run:

    ls


the shell performs the following process:

    CURRENT DIRECTORY
           |
           v
    READ DIRECTORY CONTENTS
           |
           v
    FIND FILES AND DIRECTORIES
           |
           v
    DISPLAY THEIR NAMES


Example:

    blog/
    |
    +-- 2014/
    +-- 2015/
    +-- hardware.txt


Run:

    ls


Output:

    2014    2015    hardware.txt

===============================================================================
8. CURRENT DIRECTORY
===============================================================================

The basic:

    ls

command lists the contents of the directory you are currently inside.


Example:

Current location:

    /home/amaury/blog


Filesystem:

    blog/
    |
    +-- 2014/
    +-- 2015/
    +-- hardware.txt


Command:

    ls


Output:

    2014    2015    hardware.txt


If you move into another directory:

    cd 2015


and then run:

    ls


the output will show the contents of:

    2015/

not:

    blog/

===============================================================================
9. LS AND FILESYSTEM NAVIGATION
===============================================================================

A useful command-line workflow is:

    pwd

        ↓

    WHERE AM I?


    ls

        ↓

    WHAT IS HERE?


    cd directory

        ↓

    MOVE INTO DIRECTORY


Example:

    pwd


Output:

    /home/amaury/blog


Then:

    ls


Output:

    2014    2015    hardware.txt


Then:

    cd 2015


You have now moved into:

    /home/amaury/blog/2015

===============================================================================
16. KEY TAKEAWAYS
===============================================================================

- A command is an instruction given to the computer.

- `ls` lists the contents of a directory.

- The name `ls` can be remembered as "list."

- The first character in `ls` is the lowercase letter `l`, not the number `1`.

- The `$` symbol commonly represents the shell prompt.

- The shell prompt means the terminal is ready to accept a command.

- Running `ls` displays files and directories inside the current directory.

- Directories and files can both appear in `ls` output.

- `ls` is one of the foundational commands for navigating a Unix/Linux
  filesystem.
