# Introduction: Improved Styling with CSS

## Overview

In this unit, you will build upon your foundational CSS knowledge by exploring more advanced styling techniques and navigation design.

The goal is to deepen your understanding of CSS so you can create websites that are more visually appealing, easier to navigate, and provide a better user experience.

In addition to improving the appearance of web pages, you will learn how to design effective navigation components that help users move through a website with ease.

---

## Learning Objectives

By the end of this unit, you will be able to:

* Apply a wider variety of custom colors to web pages.
* Use custom fonts to improve typography and readability.
* Style navigation elements such as links and buttons.
* Create secondary navigation using breadcrumb navigation.
* Build more polished and professional-looking websites using CSS.

---

## Topics Covered

This unit introduces several intermediate CSS concepts, including:

### Custom Colors

Learn how to use CSS to create more visually appealing color schemes by applying:

* Named colors
* Hexadecimal (Hex) colors
* RGB and RGBA colors
* HSL and HSLA colors

Understanding color systems allows you to create consistent, accessible, and attractive website designs.

---

### Typography

Improve the appearance of text by working with fonts and typography.

Topics include:

* Font families
* Font sizing
* Font weight
* Font styles
* Font fallback strategies
* Readability and accessibility

Good typography improves both aesthetics and user experience.

---

### Navigation Design

Navigation is one of the most important components of any website.

You will learn how to style common navigation elements, including:

* Hyperlinks (`<a>`)
* Navigation menus
* Buttons
* Hover effects
* Active and visited link states

Well-designed navigation helps users quickly locate information throughout a website.

---

### Breadcrumb Navigation

You will also learn how to create **breadcrumb navigation**, a secondary navigation system that displays a user's current location within a website.

Example:

```text
Home > Products > Electronics > Laptops
```

Breadcrumbs help users:

* Understand where they are within a website.
* Navigate back to previous pages.
* Improve overall usability.
* Enhance the user experience on larger websites.

---

## Why This Matters

CSS is responsible for the presentation layer of a website.

As your CSS skills improve, you gain greater control over:

* Layout
* Colors
* Typography
* Navigation
* Visual hierarchy
* User experience (UX)

Mastering these concepts enables you to build websites that are both functional and visually professional.

---

## Key Takeaways

In this unit, you will move beyond the basics of CSS by learning intermediate styling techniques that enhance the appearance and usability of websites.

You will focus on:

* Applying custom colors
* Improving typography
* Styling navigation elements
* Creating breadcrumb navigation

These skills will help you create websites that are cleaner, more attractive, and easier for users to navigate.

> **Remember:** Great web design is more than appearance—it combines thoughtful styling, intuitive navigation, accessibility, and a positive user experience.
