# Full Stack Engineer Journey – Codecademy

## Introduction

This repository contains my notes, exercises, and projects from the **Codecademy Full-Stack Engineer Career Path**. The purpose of this repo is to document my learning journey as I develop the skills required to build modern web applications from the ground up.

Throughout this course I will be learning how both the **front-end** and **back-end** of web applications work together. The material covered includes fundamental web technologies, programming concepts, databases, and server-side development.

## What This Repository Contains

This repository is organized as a personal knowledge base and practice environment. It includes:

- **Structured notes** from lessons and documentation
- **Code examples** demonstrating key programming concepts
- **Practice exercises** and small projects
- **Reference material** for important commands, syntax, and workflows

## Technologies Covered

During this course I will be working with technologies such as:

- **HTML** – Structuring web content  
- **CSS** – Styling and layout  
- **JavaScript** – Client-side programming

- **Node.js** – Server-side JavaScript runtime  
- **Express.js** – Backend web framework  
- **APIs** – Communication between systems  
- **Databases** – Data storage and retrieval  
- **Full-Stack Development Concepts**
